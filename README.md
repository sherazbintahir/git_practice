# lets start !

This is a simple project to learn how to use git and github.

# first we need to initialize a git repository in our project folder:
U = untracked file
A = added file
M = modified file
D = deleted file
R = renamed file
 
# Check Logs 
1. check the logs : git log --oneline
2. check the logs with details : git log
3. check the logs with more details : git log -p

# Git Status
1. check stage of a file: git status -s 
2. check all changes: git status
